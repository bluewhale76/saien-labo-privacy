菜園Labo プライバシーポリシー

最終更新日：2026年6月16日

【収集する情報について】
菜園Laboは、ユーザーが入力した栽培記録・タスク・写真などのデータをすべてお使いの端末内にのみ保存します。
当アプリが個人情報やデータを外部のサーバーへ送信することはありません。

【第三者への提供】
当アプリは、いかなる個人情報も第三者に提供・販売しません。

【広告・アナリティクスについて】
当アプリは広告の表示およびアクセス解析ツールを使用していません。
